HTML Tags

Table Tags

Tag Name Description
table Table Element The wrapper element for all HTML tables.
thead Table Head Element The set of rows defining the column headers in a table.
tbody Table Body Element The set of rows containing actual table data.
tr Table Row Element The table row element defines a row of cells in a table.
th Table Head Element The table header container
td Table Data Element The table data container

Table attibutes

Atribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.
